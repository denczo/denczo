I like turning ideas into things you can hear, see and touch — synthesizers,
reactive lighting, little robots and the odd 3D-printed gadget. Some of it is
serious, most of it is for the fun of building.

### 🎹 Audio & DSP
- **[pyblaster](https://github.com/denczo/pyblaster)** — monophonic synthesizer with MIDI, ADSR, reverb & LFO ⭐ 17
- **[monomind](https://github.com/denczo/monomind)** — monophonic synth with a 32-step sequencer for bass lines
- **[FM_Plotter](https://github.com/denczo/FM_Plotter)** — interactive tool for learning frequency modulation
- **[FREM](https://github.com/denczo/FREM)** — Android app that teaches FM synthesis, hands-on

### 💡 Lighting & Hardware
- **[nanoleaf-reloaded](https://github.com/denczo/nanoleaf-reloaded)** — Home Assistant integration for per-panel Nanoleaf control
- **[ambileaf](https://github.com/denczo/ambileaf)** — ambilight that syncs your Nanoleafs to whatever's on screen
- **[arcreactor](https://github.com/denczo/arcreactor)** — animation code for a 3D-printed, illuminated Arc Reactor
- **[pipboy_watchface](https://github.com/denczo/pipboy_watchface)** — Pip-Boy watchface for the Mi Band 8

### 🤖 ML & Robotics
- **[RaspberryBot](https://github.com/denczo/RaspberryBot)** — line-follower robot that learns with Q-learning
- **[optimization-algos](https://github.com/denczo/optimization-algos)** — hill climbing, simulated annealing & genetic algorithm
- **[ConvAutoencoderRPI](https://github.com/denczo/ConvAutoencoderRPI)** — convolutional autoencoder (bachelor thesis)

### 🌐 Web & Tooling
- **[kivy_examples](https://github.com/denczo/kivy_examples)** — a learning collection for the Kivy UI framework ⭐ 15
- **[nextfolio](https://github.com/denczo/nextfolio)** — responsive portfolio site built with Next.js
- **[fruitinator](https://github.com/denczo/fruitinator)** — responsive e-commerce frontend in React
